# Как скачать и запустить MultiMonitorPanel

Эта инструкция для обычного пользователя Windows, который не собирает программы
из исходников и просто хочет запустить MultiMonitorPanel.

## Коротко

1. Откройте [последний релиз](https://github.com/e-u-shapovalov/MultiMonitorPanel/releases/latest).
2. Найдите блок **Assets**.
3. Скачайте **`panel.exe`**.
4. Положите файл в удобную папку.
5. Запустите двойным кликом.

## Что именно скачивать

В текущем релизе готовая программа лежит в Assets как:

```text
panel.exe
```

Это portable-версия: установщик не нужен.

Если в будущих релизах появится архив, например
`MultiMonitorPanel-v1.0.1-win64.zip`, скачивайте его из того же блока **Assets**
и распаковывайте перед запуском.

## Что не скачивать

Если вы обычный пользователь:

- не нажимайте **Code -> Download ZIP** на главной странице репозитория;
- не скачивайте **Source code (zip)**;
- не скачивайте **Source code (tar.gz)**.

Это исходный код. Он нужен разработчикам, а не для обычного запуска программы.

## Где находится GitHub Releases

GitHub Releases - это страница готовых версий проекта.

Открыть её можно так:

1. Зайдите на страницу репозитория.
2. Найдите справа или сверху ссылку **Releases**.
3. Откройте самый свежий релиз.
4. Внизу описания найдите **Assets**.
5. Скачайте готовый файл программы.

Самая простая ссылка:

<https://github.com/e-u-shapovalov/MultiMonitorPanel/releases/latest>

## Установка

MultiMonitorPanel не устанавливается в систему. Его достаточно положить в папку.

Рекомендуемый вариант:

```text
C:\Tools\MultiMonitorPanel\panel.exe
```

Можно использовать и другую папку, например:

```text
D:\Programs\MultiMonitorPanel\panel.exe
```

Не запускайте программу прямо из временной папки браузера или архива. Если
будущий релиз будет распространяться zip-архивом, сначала распакуйте его.

## Первый запуск

1. Дважды щёлкните по `panel.exe`.
2. Внизу каждого монитора появится панель.
3. Нажмите правой кнопкой мыши по панели.
4. Выберите **Изменить настройки...**.
5. Добавьте свои кнопки: программы, папки, консоли, админские утилиты.

При первом запуске программа может создать рядом с собой:

```text
info.txt
lang\
ico\
```

Это нормально. `info.txt` - краткая справка, `lang` - языковые файлы,
`ico` - папка для ваших иконок.

## Автозапуск вместе с Windows

1. Нажмите `Win+R`.
2. Введите:

   ```text
   shell:startup
   ```

3. Нажмите Enter.
4. Создайте ярлык на `panel.exe`.
5. Перетащите ярлык в открывшуюся папку автозагрузки.

## Если Windows показывает SmartScreen

Windows может предупредить, что приложение неизвестно. Это часто происходит с
новыми open-source программами без коммерческой подписи.

Проверьте, что файл скачан отсюда:

```text
https://github.com/e-u-shapovalov/MultiMonitorPanel
```

Если адрес верный, нажмите:

```text
Подробнее -> Выполнить в любом случае
```

## Если панель не появилась

- Проверьте, что `panel.exe` запущен.
- Откройте Диспетчер задач и посмотрите, нет ли уже работающего `panel.exe`.
- Закройте процесс и запустите снова.
- Если кнопок нет, нажмите правой кнопкой по полосе и выберите **Образцовый набор**.

## Для разработчиков

Разработчикам можно скачать исходный код через Git:

```bat
git clone https://github.com/e-u-shapovalov/MultiMonitorPanel.git
cd MultiMonitorPanel
build.bat
```

Но для обычного пользователя Git, Visual Studio и сборка не нужны.
