# Скриншоты для README

В эту папку стоит добавить релизные изображения MultiMonitorPanel. Скриншоты
нужны не для сборки, а для GitHub, Яндекса, Google и пользователей, которые
хотят быстро понять внешний вид программы.

Уже добавлено:

```text
multi-monitor-panel-windows-ru.png
```

Ещё желательно добавить:

```text
settings-editor-ru.png
context-menu.png
```

Что показать:

- `multi-monitor-panel-windows-ru.png` - нижняя панель MultiMonitorPanel в
  реальной рабочей среде Windows;
- `settings-editor-ru.png` - окно **Изменить настройки...** с несколькими
  кнопками, разделителем и разными краями;
- `context-menu.png` - меню правого клика с пунктами настройки, языка и выхода.

Требования:

- PNG;
- без персональных данных, токенов, путей к приватным папкам и названий клиентов;
- ширина 1200-1800 px для основного скриншота;
- русский интерфейс для главного README, английский можно добавить отдельно как
  `settings-editor-en.png`.
